Glossary

Abandon
To turn a face-up ongoing scheme card face down and put it on the bottom of its owner's scheme deck. See rule <a href=mtgjudge://701.23>701.23</a>, "Abandon."

Ability
1. Text on an object that explains what that object does or can do.
2. An activated or triggered ability on the stack. This kind of ability is an object.
See rule <a href=mtgjudge://112>112</a>, "Abilities," and section 6, "Spells, Abilities, and Effects."

Ability Word
An italicized word with no rules meaning that ties together abilities on different cards that have similar functionality. See rule <a href=mtgjudge://207.2c>207.2c</a>.

Absorb
A keyword ability that prevents damage. See rule <a href=mtgjudge://702.62>702.62</a>, "Absorb."

Activate
To put an activated ability onto the stack and pay its costs, so that it will eventually resolve and have its effect. See rule <a href=mtgjudge://602>602</a>, "Activating Activated Abilities."

Activated Ability
A kind of ability. Activated abilities are written as "[Cost]: [Effect.] [Activation instructions (if any).]" See rule <a href=mtgjudge://112>112</a>, "Abilities," and rule <a href=mtgjudge://602>602</a>, "Activating Activated Abilities."

Activation Cost
Everything that appears before the colon in an activated ability's text. It must be paid to activate the ability. See rule <a href=mtgjudge://117>117</a>, "Costs," and rule <a href=mtgjudge://602>602</a>, "Activating Activated Abilities."

Active Player
The player whose turn it is. See rule <a href=mtgjudge://102.1>102.1</a>.

Active Player, Nonactive Player Order
A system that determines the order by which players make choices if multiple players are instructed to make choices at the same time. See rule <a href=mtgjudge://101.4>101.4</a>. This rule is modified for games using the shared team turns option; see rule <a href=mtgjudge://805.6>805.6</a>.

Active Team
The team whose turn it is in a game using the shared team turns option. See rule <a href=mtgjudge://805.4a>805.4a</a>.

Additional Cost
A cost a spell may have that its controller may pay (or, in some cases, must pay) in addition to its mana cost to cast that spell. See rule <a href=mtgjudge://117>117</a>, "Costs," and rule <a href=mtgjudge://601>601</a>, "Casting Spells."

Affinity
A keyword ability that reduces how much mana you need to spend to cast a spell. See rule <a href=mtgjudge://702.39>702.39</a>, "Affinity."

Alternating Teams Variant
A multiplayer variant played among two or more teams of equal size. See rule <a href=mtgjudge://811>811</a>, "Alternating Teams Variant."

Alternative Cost
A cost a spell may have that its controller can pay rather than paying its mana cost. See rule <a href=mtgjudge://117>117</a>, "Costs," and rule <a href=mtgjudge://601>601</a>, "Casting Spells."

Amplify
A keyword ability than can have a creature enter the battlefield with +1/+1 counters on it. See rule <a href=mtgjudge://702.36>702.36</a>, "Amplify."

Annihilator
A keyword ability that can make a creature particularly brutal when it attacks. See rule <a href=mtgjudge://702.84>702.84</a>, "Annihilator."

Ante
1. A zone used only when playing "for keeps."
2. To put a card into the ante zone.
See rule <a href=mtgjudge://407>407</a>, "Ante."

APNAP Order
See Active Player, Nonactive Player Order.

Archenemy
1. A casual variant in which a team of players faces off against a single opponent bolstered with powerful scheme cards. See rule <a href=mtgjudge://904>904</a>, "Archenemy."
2. A player in an Archenemy game who is playing with a scheme deck.

Artifact
A card type. An artifact is a permanent. See rule <a href=mtgjudge://301>301</a>, "Artifacts."

Artifact Creature
A combination of artifact and creature that's subject to the rules for both. See rule <a href=mtgjudge://301>301</a>, "Artifacts," and rule <a href=mtgjudge://302>302</a>, "Creatures."

Artifact Land
A combination of artifact and land that's subject to the rules for both. Artifact lands can only be played as lands, not cast as spells. See rule <a href=mtgjudge://301>301</a>, "Artifacts," and rule <a href=mtgjudge://305>305</a>, "Lands."

Artifact Type
A subtype that's correlated to the artifact card type. See rule <a href=mtgjudge://301>301</a>, "Artifacts." See rule <a href=mtgjudge://205.3g>205.3g</a> for the list of artifact types.

As Though
Text used to indicate that the game, for some specific purpose, treats a condition as true even though it's not. See rule <a href=mtgjudge://609.4>609.4</a>.

Assign Combat Damage
To determine how an attacking or blocking creature will deal its combat damage. See rule <a href=mtgjudge://510>510</a>, "Combat Damage Step."

At End of Turn (Obsolete)
A trigger condition printed on abilities that triggered at the beginning of the end step (which is not the last thing to happen in the turn). Cards that were printed with that text have received errata in the Oracle card reference to say "at the beginning of the end step" or "at the beginning of the next end step." See rule <a href=mtgjudge://513>513</a>, "End Step."

Attach
To move an Aura, Equipment, or Fortification onto another object. See rule <a href=mtgjudge://701.3>701.3</a>, "Attach."

Attack
To send a creature into combat offensively. A creature can attack a player or a planeswalker. See rule <a href=mtgjudge://508>508</a>, "Declare Attackers Step."

Attack Alone
A creature "attacks alone" if it's the only creature declared as an attacker during the declare attackers step. A creature "is attacking alone" if it's attacking but no other creatures are. See rule <a href=mtgjudge://506.5>506.5</a>.

Attack Left Option
An option that may be used in certain multiplayer variants. See rule <a href=mtgjudge://803>803</a>, "Attack Left and Attack Right Options."

Attack Multiple Players Option
An option that may be used in certain multiplayer variants. See rule <a href=mtgjudge://802>802</a>, "Attack Multiple Players Option."

Attack Right Option
An option that may be used in certain multiplayer variants. See rule <a href=mtgjudge://803>803</a>, "Attack Left and Attack Right Options."

Attacking Creature
A creature that has either been declared as part of a legal attack during the combat phase (once all costs to attack, if any, have been paid), or a creature that has been put onto the battlefield attacking. It remains an attacking creature until it's removed from combat or the combat phase ends, whichever comes first. See rule <a href=mtgjudge://508>508</a>, "Declare Attackers Step."

Attacks and Isn't Blocked
An ability that triggers when a creature "attacks and isn't blocked" triggers when the creature becomes an unblocked attacking creature. See rule <a href=mtgjudge://509.1h>509.1h</a>.

Aura
An enchantment subtype. Aura spells target objects or players, and Aura permanents are attached to objects or players. See rule <a href=mtgjudge://303>303</a>, "Enchantments," and rule <a href=mtgjudge://702.5>702.5</a>, "Enchant."

Aura Swap
A keyword ability that lets you exchange an Aura on the battlefield with one in your hand. See rule <a href=mtgjudge://702.63>702.63</a>, "Aura Swap."

Banding, "Bands with Other"
Banding is a keyword ability that modifies the rules for declaring attackers and assigning combat damage. "Bands with other" is a specialized version of the ability. See rule <a href=mtgjudge://702.20>702.20</a>, "Banding."

Basic
A supertype that's normally relevant on lands. Any land with this supertype is a basic land. See rule <a href=mtgjudge://205.4>205.4</a>, "Supertypes."

Basic Landcycling
See Typecycling.

Basic Land Type
There are five "basic land types": Plains, Island, Swamp, Mountain, and Forest. Each one has a mana ability associated with it. See rule <a href=mtgjudge://305>305</a>, "Lands."

Battle Cry
A keyword ability that makes other attacking creatures better in combat. See rule <a href=mtgjudge://702.89>702.89</a>, "Battle Cry."

Battlefield
A zone. The battlefield is the zone in which permanents exist. It used to be known as the "in-play" zone. See rule <a href=mtgjudge://403>403</a>, "Battlefield."

Becomes
A word used in some trigger events to indicate a change in status or characteristics. See rule <a href=mtgjudge://603.2d>603.2d</a>.

Beginning of Combat Step
Part of the turn. This step is the first step of the combat phase. See rule <a href=mtgjudge://507>507</a>, "Beginning of Combat Step."

Beginning Phase
Part of the turn. This phase is the first phase of the turn. See rule <a href=mtgjudge://501>501</a>, "Beginning Phase."

Block
To send a creature into combat defensively. A creature can block an attacking creature. See rule <a href=mtgjudge://509>509</a>, "Declare Blockers Step."

Block Alone
A creature "blocks alone" if it's the only creature declared as a blocker during the declare blockers step. A creature "is blocking alone" if it's blocking but no other creatures are. See rule <a href=mtgjudge://506.5>506.5</a>.

Blocked Creature
An attacking creature that another creature blocks or that an effect causes to become blocked. It remains a blocked creature until it's removed from combat, an effect says that it becomes unblocked, or the combat phase ends, whichever comes first. See rule <a href=mtgjudge://509>509</a>, "Declare Blockers Step."

Blocking Creature
A creature that has either been declared as part of a legal block during the combat phase (once all costs to block, if any, have been paid), or a creature that has been put onto the battlefield blocking. It remains a blocking creature until it's removed from combat or the combat phase ends, whichever comes first. See rule <a href=mtgjudge://509>509</a>, "Declare Blockers Step."

Bloodthirst
A keyword ability that can have a creature enter the battlefield with +1/+1 counters on it. See rule <a href=mtgjudge://702.52>702.52</a>, "Bloodthirst."

Bury (Obsolete)
A term that meant "put [a permanent] into its owner's graveyard." In general, cards that were printed with the term "bury" have received errata in the Oracle card reference to read, "Destroy [a permanent]. It can't be regenerated," or "Sacrifice [a permanent]."

Bushido
A keyword ability that can make a creature better in combat. See rule <a href=mtgjudge://702.43>702.43</a>, "Bushido."

Buyback
A keyword ability of instants and sorceries that can let the spell return to its owner's hand as it resolves. See rule <a href=mtgjudge://702.25>702.25</a>, "Buyback."

Card
The standard component of the game. _Magic_ cards may be traditional or nontraditional. Tokens aren't considered cards. In the text of spells or abilities, the term "card" is used only to refer to a card that's not on the battlefield or on the stack, such as a creature card in a player's hand. See rule <a href=mtgjudge://108>108</a>, "Cards."

Card Type
A characteristic. Except for abilities on the stack, each object has a card type, even if that object isn't a card. Each card type has its own rules. See rule <a href=mtgjudge://205>205</a>, "Type Line," and section 3, "Card Types."

Cascade
A keyword ability that may let a player cast a random extra spell for no cost. See rule <a href=mtgjudge://702.83>702.83</a>, "Cascade."

Cast
To take a spell from where it is (usually the hand), put it on the stack, and pay its costs, so that it will eventually resolve and have its effect. See rule <a href=mtgjudge://601>601</a>, "Casting Spells."

Caster (Obsolete)
An obsolete term that referred to the player who cast a spell. In general, cards that were printed with the term "caster" have received errata in the Oracle card reference to say "controller."

Casting Cost (Obsolete)
An obsolete term for mana cost. Cards printed with this text have received errata in the Oracle card reference.

Champion, Championed
"Champion" is a keyword ability that lets one creature temporarily replace another. A permanent is "championed" by another permanent if the latter exiles the former as the direct result of a champion ability. See rule <a href=mtgjudge://702.70>702.70</a>, "Champion."

Change a Target
To choose a new, legal target for a spell or ability. See rule <a href=mtgjudge://114.6>114.6</a>.

Changeling
A characteristic-defining ability that grants the object it's on every creature type. See rule <a href=mtgjudge://702.71>702.71</a>, "Changeling."

Chaos Ability
An ability of a plane card that triggers "Whenever you roll {C}" on the planar die in the Planechase casual variant. See rule <a href=mtgjudge://309.7>309.7</a>.

Chaos Symbol
The chaos symbol {C} appears on the planar die and in some triggered abilities of plane cards in the Planechase casual variant. See rule <a href=mtgjudge://107.12>107.12</a>.

Characteristics
Information that defines an object. See rule <a href=mtgjudge://109.3>109.3</a>.

Characteristic-Defining Ability
A kind of static ability that conveys information about an object's characteristics that would normally be found elsewhere on that object (such as in its mana cost, type line, or power/toughness box). See rule <a href=mtgjudge://604.3>604.3</a>.

Clash
To have a mini-contest involving the top cards of players' libraries. See rule <a href=mtgjudge://701.20>701.20</a>, "Clash."

Cleanup Step
Part of the turn. This step is the second and final step of the ending phase. See rule <a href=mtgjudge://514>514</a>, "Cleanup Step."

Collector Number
A number printed on most cards that has no effect on game play. See rule <a href=mtgjudge://212>212</a>, "Information Below the Text Box."

Color
1. A characteristic of an object. See rule <a href=mtgjudge://105>105</a>, "Colors," and rule <a href=mtgjudge://202>202</a>, "Mana Cost and Color."
2. An attribute mana may have. See rule <a href=mtgjudge://106>106</a>, "Mana."

Colorless
1. An object with no color is colorless. Colorless is not a color. See rule <a href=mtgjudge://105>105</a>, "Colors," and rule <a href=mtgjudge://202>202</a>, "Mana Cost and Color."
2. A type of mana. See rule <a href=mtgjudge://106>106</a>, "Mana," and rule <a href=mtgjudge://107.4c>107.4c</a>.

Color Identity
A set of colors that determines what cards may be included in a deck for the Commander casual variant. See rule <a href=mtgjudge://903.4>903.4</a>.

Color Indicator
A characteristic of an object. See rule <a href=mtgjudge://105>105</a>, "Colors," and rule <a href=mtgjudge://204>204</a>, "Color Indicator."

Combat Damage
Damage dealt during the combat damage step by attacking creatures and blocking creatures as a consequence of combat. See rule <a href=mtgjudge://510>510</a>, "Combat Damage Step."

Combat Damage Step
Part of the turn. This step is the fourth step of the combat phase. See rule <a href=mtgjudge://510>510</a>, "Combat Damage Step."

Combat Phase
Part of the turn. This phase is the third phase of the turn. See rule <a href=mtgjudge://506>506</a>, "Combat Phase."

Command
A zone for certain specialized objects that have an overarching effect on the game, yet are not permanents and cannot be destroyed. See rule <a href=mtgjudge://408>408</a>, "Command."

Commander
1. A casual variant in which each deck is led by a legendary creature. See rule <a href=mtgjudge://903>903</a>, "Commander."
2. A designation given to one legendary creature card in each player's deck in the Commander casual variant.

Concede
To quit the game. Conceding a game immediately causes that player to leave that game and lose that game. See rule <a href=mtgjudge://104>104</a>, "Ending the Game."

Conspire
A keyword ability that creates a copy of a spell. See rule <a href=mtgjudge://702.76>702.76</a>, "Conspire."

Constructed
A way of playing in which each player creates his or her own deck ahead of time. See rule <a href=mtgjudge://100.2a>100.2a</a>.

Continuous Effect
An effect that modifies characteristics of objects, modifies control of objects, or affects players or the rules of the game, for a fixed or indefinite period. See rule <a href=mtgjudge://611>611</a>, "Continuous Effects."

Continuous Artifact (Obsolete)
An obsolete term that appeared on the type line of artifacts without activated abilities. Cards printed with this text have received errata in the Oracle card reference to simply say "Artifact."

Control, Controller
"Control" is the system that determines who gets to use an object in the game. An object's "controller" is the player who currently controls it. See rule <a href=mtgjudge://108.4>108.4</a>.

Control Another Player
To make all choices and decisions that player is allowed to make, or is told to make, by rules or by any objects. See rule <a href=mtgjudge://712>712</a>, "Controlling Another Player."

Converted Mana Cost
The total amount of mana in a mana cost, regardless of color. See rule <a href=mtgjudge://202.3>202.3</a>.

Convoke
A keyword ability that reduces how much mana you need to spend to cast a spell. See rule <a href=mtgjudge://702.49>702.49</a>, "Convoke."

Copiable Values
Values of an object's characteristics that are checked by copy effects. See rules <a href=mtgjudge://706.2>706.2</a> and <a href=mtgjudge://706.3>706.3</a>.

Copy
1. To create a new object whose copiable values have been set to those of another object.
2. An object whose copiable values have been set to those of another object.
See rule <a href=mtgjudge://706>706</a>, "Copying Objects."

Cost
An action or payment necessary to take another action or to stop another action from taking place. See rule <a href=mtgjudge://117>117</a>, "Costs."

Counter
1. To cancel a spell or ability so it doesn't resolve and none of its effects occur. See rule <a href=mtgjudge://701.5>701.5</a>, "Counter."
2. A marker placed on an object or player that modifies its characteristics or interacts with a rule or ability. See rule <a href=mtgjudge://121>121</a>, "Counters."

Counts As (Obsolete)
Some older cards were printed with text stating that the card "counts as" something. Cards printed with this text have received errata in the Oracle card reference to state that the card actually is that thing.

Creature
A card type. A creature is a permanent. See rule <a href=mtgjudge://302>302</a>, "Creatures."

Creature Type
A subtype that's correlated to the creature card type and the tribal card type. See rule <a href=mtgjudge://302>302</a>, "Creatures," and rule <a href=mtgjudge://308>308</a>, "Tribals." See rule <a href=mtgjudge://205.3m>205.3m</a> for the list of creature types.

Cumulative Upkeep
A keyword ability that imposes an increasing cost to keep a permanent on the battlefield. See rule <a href=mtgjudge://702.22>702.22</a>, "Cumulative Upkeep."

Cycling
A keyword ability that lets a card be discarded and replaced with a new card. See rule <a href=mtgjudge://702.27>702.27</a>, "Cycling."

Damage
Objects can deal "damage" to creatures, planeswalkers, and players. This is generally detrimental to the object or player that receives that damage. See rule <a href=mtgjudge://119>119</a>, "Damage."

Damage Assignment Order
The order, announced during the declare blockers step, that an attacking creature will assign its combat damage among the multiple creatures blocking it, or that a blocking creature will assign its combat damage among the multiple creatures it's blocking. See rules <a href=mtgjudge://509.2>509.2</a> and <a href=mtgjudge://509.3>509.3</a>.

Deal
See Damage.

Deathtouch
A keyword ability that causes damage dealt by an object to be especially effective. See rule <a href=mtgjudge://702.2>702.2</a>, "Deathtouch."

Deck
The collection of cards a player starts the game with; it becomes that player's library. See rule <a href=mtgjudge://100>100</a>, "General," and rule <a href=mtgjudge://103>103</a>, "Starting the Game."

Declare Attackers
To choose a set of creatures that will attack, declare whether each creature is attacking the defending player or a planeswalker that player controls, and pay any costs required to allow those creatures to attack. See rule <a href=mtgjudge://508.1>508.1</a>.

Declare Attackers Step
Part of the turn. This step is the second step of the combat phase. See rule <a href=mtgjudge://508>508</a>, "Declare Attackers Step."

Declare Blockers
To choose a set of creatures that will block, declare which attacking creature each creature is blocking, and pay any costs required to allow those creatures to block. See rule <a href=mtgjudge://509.1>509.1</a>.

Declare Blockers Step
Part of the turn. This step is the third step of the combat phase. See rule <a href=mtgjudge://509>509</a>, "Declare Blockers Step."

Defender
A keyword ability that prohibits a creature from attacking. See rule <a href=mtgjudge://702.3>702.3</a>, "Defender."

Defending Player
The player who can be attacked, and whose planeswalkers can be attacked, during the combat phase. See rule <a href=mtgjudge://506.2>506.2</a>. In certain multiplayer games, there may be more than one defending player; see rule <a href=mtgjudge://802>802</a>, "Attack Multiple Players Option," and rule <a href=mtgjudge://810.7>810.7</a>.

Defending Team
The team who can be attacked, and whose planeswalkers can be attacked, during the combat phase of a Two-Headed Giant game. See rule <a href=mtgjudge://810.7>810.7</a>.

Delayed Triggered Ability
An ability created by effects generated when some spells or abilities resolve, or when some replacement effects are applied, that does something later on rather than at that time. See rule <a href=mtgjudge://603.7>603.7</a>.

Delve
A keyword ability that reduces how much mana you need to spend to cast a spell. See rule <a href=mtgjudge://702.64>702.64</a>, "Delve."

Dependency
A system that may be used to determine in which order continuous effects in the same layer or sublayer are applied. See rule <a href=mtgjudge://613.7>613.7</a>. See also Timestamp Order.

Deploy Creatures Option
An option that may be used in certain multiplayer variants to pass control of creatures between teammates. See rule <a href=mtgjudge://804>804</a>, "Deploy Creatures Option."

Destroy
To move a permanent from the battlefield to its owner's graveyard. See rule <a href=mtgjudge://701.6>701.6</a>, "Destroy."

Devour
A keyword ability that can have a creature enter the battlefield with +1/+1 counters on it. See rule <a href=mtgjudge://702.80>702.80</a>, "Devour."

Dies
A creature "dies" if it is put into a graveyard from the battlefield. See rule <a href=mtgjudge://700.6>700.6</a>.

Discard
To move a card from its owner's hand to that player's graveyard. See rule <a href=mtgjudge://701.7>701.7</a>, "Discard."

Double-Faced Cards
Cards with two faces, one on each side of the card, and no _Magic_ card back. See rule <a href=mtgjudge://711>711</a>, "Double-Faced Cards."

Double Strike
A keyword ability that lets a creature deal its combat damage twice. See rule <a href=mtgjudge://702.4>702.4</a>, "Double Strike."

Draw
1. To put the top card of a player's library into his or her hand as a turn-based action or as the result of an effect that uses the word "draw." See rule <a href=mtgjudge://120>120</a>, "Drawing a Card."
2. The result of a game in which neither player wins or loses. See rule <a href=mtgjudge://104.4>104.4</a>.

Draw Step
Part of the turn. This step is the third and final step of the beginning phase. See rule <a href=mtgjudge://504>504</a>, "Draw Step."

Dredge
A keyword ability that lets a player return a card from his or her graveyard to his or her hand. See rule <a href=mtgjudge://702.50>702.50</a>, "Dredge."

During (Obsolete)
Some older cards used the phrase "during [phase], [action]." These abilities were called "phase abilities." In general, cards that were printed with phase abilities have received errata in the Oracle card reference so they have abilities that trigger at the beginning of a step or phase. "During" still appears in current card text, but only in its normal English sense and not as game terminology.

Echo
A keyword ability that imposes a cost to keep a permanent on the battlefield. See rule <a href=mtgjudge://702.28>702.28</a>, "Echo."

EDH (Obsolete)
An older name for the Commander casual variant. See rule <a href=mtgjudge://903>903</a>, "Commander."

Effect
Something that happens in the game as a result of a spell or ability. See rule <a href=mtgjudge://609>609</a>, "Effects."

Emblem
An emblem is a marker used to represent an object that has one or more abilities, but no other characteristics. See rule <a href=mtgjudge://113>113</a>, "Emblems."

Emperor
The middle player on each team in an Emperor game. See rule <a href=mtgjudge://809>809</a>, "Emperor Variant."

Emperor Variant
A multiplayer variant played among three-player teams. See rule <a href=mtgjudge://809>809</a>, "Emperor Variant."

Enchant
A keyword ability that defines what an Aura spell can target and what an Aura permanent can be attached to. See rule <a href=mtgjudge://303>303</a>, "Enchantments," and rule <a href=mtgjudge://702.5>702.5</a>, "Enchant."

Enchantment
A card type. An enchantment is a permanent. See rule <a href=mtgjudge://303>303</a>, "Enchantments." See also Aura.

Enchantment Type
A subtype that's correlated to the enchantment card type. See rule <a href=mtgjudge://303>303</a>, "Enchantments." See rule <a href=mtgjudge://205.3h>205.3h</a> for the list of enchantment types.

End of Combat Step
Part of the turn. This step is the fifth and final step of the combat phase. See rule <a href=mtgjudge://511>511</a>, "End of Combat Step."

End Step
Part of the turn. This step is the first step of the ending phase. See rule <a href=mtgjudge://513>513</a>, "End Step."

End the Turn
To "end the turn" as the result of an effect is to perform an expedited process that skips nearly everything else that would happen that turn. See rule <a href=mtgjudge://713>713</a>, "Ending the Turn."

Ending Phase
Part of the turn. This phase is the fifth and final phase of the turn. See rule <a href=mtgjudge://512>512</a>, "Ending Phase."

Enters the Battlefield
A nontoken permanent "enters the battlefield" when it's moved onto the battlefield from another zone. A token "enters the battlefield" when it's created. See rules <a href=mtgjudge://403.3>403.3</a>, <a href=mtgjudge://603.6a>603.6a</a>, <a href=mtgjudge://603.6e>603.6e</a>, and <a href=mtgjudge://614.12>614.12</a>.

Entwine
A keyword ability that lets a player choose all modes for a spell rather than just one. See rule <a href=mtgjudge://702.40>702.40</a>, "Entwine."

Epic
A keyword ability that lets a player copy a spell at the beginning of each of his or her upkeeps at the expense of casting any other spells for the rest of the game. See rule <a href=mtgjudge://702.48>702.48</a>, "Epic."

Equip
A keyword ability that lets a player attach an Equipment to a creature he or she controls. See rule <a href=mtgjudge://301>301</a>, "Artifacts," and rule <a href=mtgjudge://702.6>702.6</a>, "Equip."

Equipment
An artifact subtype. Equipment can be attached to creatures. See rule <a href=mtgjudge://301>301</a>, "Artifacts," and rule <a href=mtgjudge://702.6>702.6</a>, "Equip."

Evasion Ability
An ability that restricts what creatures can block an attacking creature. See rules <a href=mtgjudge://509.1b>509.1b</a>-c.

Event
Anything that happens in a game. See rule <a href=mtgjudge://700.1>700.1</a>.

Evoke
A keyword ability that causes a permanent to be sacrificed when it enters the battlefield. See rule <a href=mtgjudge://702.72>702.72</a>, "Evoke."

Exalted
A keyword ability that can make a creature better in combat. See rule <a href=mtgjudge://702.81>702.81</a>, "Exalted."

Exchange
To swap two things, such as objects, sets of objects, or life totals. See rule <a href=mtgjudge://701.8>701.8</a>, "Exchange."

Exile

Flying

Forecast

Forestwalk

Frenzy

Independent
See Dependency.

A card type. An instant is not a permanent. See rule <a href=mtgjudge://304>304</a>, "Instants."

Interrupt (Obsolete)

All abilities that, as printed, said a player could "play as an interrupt" can now be activated like any other activated abilities (unless they're mana abilities, in which case they follow those rules instead). Intimidate

Island

Islandcycling

Islandwalk
See Landwalk.

A card type. A land is a permanent. See rule <a href=mtgjudge://305>305</a>, "Lands."

A subtype that's correlated to the land card type. See rule <a href=mtgjudge://305>305</a>, "Lands." See rule <a href=mtgjudge://205.3i>205.3i</a> for the list of land types.

Landwalk
A generic term for a group of keyword abilities that restrict whether a creature may be blocked. See rule <a href=mtgjudge://702.13>702.13</a>, "Landwalk."

Last Known Information
Information about an object that's no longer in the zone it's expected to be in, or information about a player that's no longer in the game. See rules <a href=mtgjudge://112.7a>112.7a</a>, <a href=mtgjudge://608.2b>608.2b</a>, <a href=mtgjudge://608.2g>608.2g</a>, and <a href=mtgjudge://800.4g>800.4g</a>.

Leaves the Battlefield
A permanent "leaves the battlefield" when it's moved from the battlefield to another zone, or (if it's phased in) when it leaves the game because its owner leaves the game. See rules <a href=mtgjudge://603.6c>603.6c</a> and <a href=mtgjudge://603.6d>603.6d</a>.

Legal Text
Information printed directly below the text box that has no effect on game play. See rule <a href=mtgjudge://212>212</a>, "Information Below the Text Box."

Legend (Obsolete)
An obsolete creature type. Cards printed with this subtype have been given errata in the Oracle card reference so they have the legendary supertype instead. See Legendary.

Legendary
A supertype that's normally relevant on permanents. See rule <a href=mtgjudge://205.4>205.4</a>, "Supertypes." See also Legend Rule.

Lethal Damage

Level Symbol

Level Up

Leveler Cards
Cards with striated text boxes and three power/toughness boxes. See rule <a href=mtgjudge://710>710</a>, "Leveler Cards."

Library
1. A zone. A player's library is where that player draws cards from.
2. All the cards in a player's library.
See rule <a href=mtgjudge://401>401</a>, "Library."

Life, Life Total

Life Modifier
A characteristic that only vanguards have. See rule <a href=mtgjudge://211>211</a>, "Life Modifier."

Local Enchantment (Obsolete)

Mana Cost

The reference that contains the up-to-date wordings (in English) for all tournament-legal cards. A card's Oracle text can be found using the Gatherer card database at http://gatherer.wizards.com. See rule <a href=mtgjudge://108.1>108.1</a>.

Outside the Game
An object is "outside the game" if it isn't in any of the game's zones. See rule <a href=mtgjudge://400.10>400.10</a>.

Owner
The player who (for purposes of the game) a card, token, or copy of a spell belongs to. See rules <a href=mtgjudge://108.3>108.3</a>, <a href=mtgjudge://110.2>110.2</a>, <a href=mtgjudge://110.5a>110.5a</a>, and <a href=mtgjudge://111.2>111.2</a>.

Permanent

Placed

Vanishing
A keyword ability that limits how long a permanent remains on the battlefield. See rule <a href=mtgjudge://702.61>702.61</a>, "Vanishing."

Variant
An additional set of rules that determines the style of a multiplayer game. See rule <a href=mtgjudge://800.2>800.2</a>.

Vigilance
A keyword ability that lets a creature attack without tapping. See rule <a href=mtgjudge://702.19>702.19</a>, "Vigilance."

Wall
A creature type with no particular rules meaning. Older cards with the Wall creature type but without defender had an unwritten ability that precluded them from attacking. Those cards have received errata in the Oracle card reference to have defender. Some older cards that referenced the Wall creature type have also received errata. See Defender.

Win the Game
There are several ways to win the game. See rule <a href=mtgjudge://104>104</a>, "Ending the Game," and rules <a href=mtgjudge://810.8>810.8</a> (for additional rules for Two-Headed Giant games) and rule <a href=mtgjudge://809.5>809.5</a> (for additional rules for Emperor games).

Wither
A keyword ability that affects how an object deals damage to a creature. See rule <a href=mtgjudge://702.78>702.78</a>, "Wither."

Wizardcycling
See Typecycling.

World
A supertype that's normally relevant on enchantments. See rule <a href=mtgjudge://205.4>205.4</a>, "Supertypes." See also World Rule.

World Rule
A state-based action that causes all permanents with the world supertype except the newest to be put into their owners' graveyards. See rule <a href=mtgjudge://704.5m>704.5m</a>.

X
A placeholder for a number that needs to be determined. See rule <a href=mtgjudge://107.3>107.3</a>.

Y
See X.

You, Your
Words that refer to an object's controller, its would-be controller (if a player is attempting to cast or activate it), or its owner (if it has no controller). See rule <a href=mtgjudge://109.5>109.5</a>.

Zone
A place where objects can be during a game. See section 4, "Zones."

Zone-Change Triggers
Trigger events that involve objects changing zones. See rule <a href=mtgjudge://603.6>603.6</a>.
